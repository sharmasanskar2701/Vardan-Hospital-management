# Vardan-Hospital-management
It allows doctors to create and manage patient prescriptions online, enabling patients to easily view, download, and access their medical prescriptions anytime. This system improves accuracy, reduces paperwork, and enhances overall hospital efficiency.
This repository contains the source code of the *Vardan Hospital Project* developed as an academic project.

---

## Project Description

This project is created to demonstrate a basic hospital-related web application.  
It includes simple pages and functionality required for a hospital project.
